Datos en Tiempo Real de los Terremotos:

Esta aplicación web interactiva, desarrollada con Streamlit, permite visualizar eventos sísmicos en tiempo real en Puerto Rico y a nivel mundial. Los datos se obtienen dinámicamente mediante la librería quakefeeds y se presentan de forma clara y visual.

Funcionalidades principales:

Filtros por severidad, período de tiempo y zona geográfica

Visualización en mapa interactivo con Plotly y Mapbox

Histogramas de magnitud y profundidad

Tabla dinámica de eventos sísmicos

Estadísticas resumidas en tiempo real

Tecnologías utilizadas

Python

Streamlit

Pandas

Plotly

QuakeFeeds


Aplicación desarrollada como parte del curso INGE3016 en la Universidad de Puerto Rico en Humacao.
